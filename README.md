    │       └── 00000000.jpg ~ 00000799.jpg + 00000920.jpg ~ 00001052.jpg

    │   ├── img
    │   │   └── 00000800.jpg ~ 00000919.jpg
    │   └── label
    │       └── 00000800.jpg ~ 00000919.jpg

    │

    │
    ├── postprocess.py (For postprocessing prediction to final result)
